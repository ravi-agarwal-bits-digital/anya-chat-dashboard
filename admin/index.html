Security notes
1
Excel is encrypted before publishingThe dashboard decrypts it after password login using the same passphrase. The encrypted file keeps the legacy AANYAENC1 signature for compatibility.
2
Token remains session-onlyThe token is never persisted in localStorage and clears when the browser session ends.
3
Use fine-grained GitHub tokenGive access only to this repository and contents write permission. Do not use a broad classic personal access token.
4
Only the live workbook is retainedEach successful publish replaces the encrypted live workbook and updates dashboard-config.json; dated workbook backups are not created.
